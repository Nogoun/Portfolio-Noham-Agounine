Résumé du projet

Installation de Linux Debian sur une machine virtuelle

J'ai réalisé un projet consistant à installer la distribution Linux Debian sur une machine virtuelle. L'objectif était de créer un environnement virtuel pour pouvoir travailler avec Linux sans modifier mon système d'exploitation principal.

Exercices de TP pour guider l'installation

Pour réaliser cette installation, nous avons été guidés par deux exercices de TP. Ces exercices fournissaient les instructions étape par étape pour installer Debian sur une machine virtuelle, en nous aidant à configurer les paramètres nécessaires.

Installation de Snap et Flatpak

Après avoir installé Debian sur la machine virtuelle, j'ai procédé à l'installation de Snap et Flatpak. Snap et Flatpak sont des gestionnaires de paquets universels qui permettent d'installer facilement des applications Linux sur différentes distributions.
Ils offrent une méthode pratique pour obtenir des logiciels à partir de sources tierces.

Installation de packages Debian

En plus de Snap et Flatpak, j'ai installé plusieurs packages Debian essentiels pour mon environnement de développement. Cela incluait le JDK (Java Development Kit), Git pour le contrôle de version et l'IDE Netbeans pour le développement Java.
Ces packages sont couramment utilisés dans le développement logiciel et m'ont permis de mettre en place un environnement de travail complet.

Conclusion

En résumé, ce projet m'a permis d'acquérir des compétences en installation de Linux Debian sur une machine virtuelle, ainsi que de familiariser avec les gestionnaires de paquets Snap et Flatpak.
J'ai également pu configurer mon environnement de développement en installant des packages Debian tels que JDK,
Git et Netbeans. Cette expérience a renforcé ma compréhension du système d'exploitation Linux et m'a préparé à travailler sur des projets de développement dans cet environnement.
